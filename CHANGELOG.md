# Changelog

## v1.2.1 (2026-07-12)

### Bug Fixes
- 修复手机发送文件上传链路中断（sendNext 未调用）
- 修复手机发送页面等待/拒收时进度条仍显示
- 修复接收记录批次分组不可靠（改为基于 /upload-start 信号）
- 修复自动接收和开机自启开关状态管理（State 类型统一为 Arc<AppState>）
- 修复开机自启命令缺失（get_autostart/set_autostart 未实现）
- 修复发送完成计数显示错误（已发送 1/0）
- 修复发送页文件数组未清理导致批量混乱
- 修复 mobile_rejected 翻译缺失
- 修复设置子页面进入动画卡顿（改为并行加载）

### Features
- 右键文件分享冷启动自动弹二维码
- 运行中再次点击 exe 自动置顶已有窗口
- 右键菜单注册状态持久化（%LOCALAPPDATA%\Su\ctx_registered.cfg）
- 手机发送页移除"返回重选"按钮（与继续发送重复）
- 分享中选择文件后自动弹出二维码

### Improvements
- 设置页面拆分为独立文件，子页面动态加载
- 补全开机自启和自动销毁命令注册
- 分享中启动时不记录（关闭清空）
